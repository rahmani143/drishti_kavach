# ⚡ QUICK START GUIDE - 5 MINUTE SETUP

## What This Does
Convert your images → .pt file → Train ML model → Get predictions

```bash

pip install torch torchvision Pillow numpy
```

## 🚀 FASTEST WAY - Run Everything at Once

If you want to run all 3 steps automatically:

```bash

```

This will:
1. ✅ Create dataset.pt from your images
2. ✅ Load and prepare data
3. ✅ Train a model
4. ✅ Save trained model
5. ✅ Run inference

## 📝 STEP-BY-STEP WAY - Run Each Step Manually

### Step 1: Create .PT File (1-5 minutes)

```bash
python step1_create_dataset_pt.py
```

**What happens:**
- Loads all images from `dataset/train/images` and `dataset/valid/images`
- Resizes to 416x416 pixels
- Converts to PyTorch tensors
- Saves as `my_dataset.pt` (100-300 MB depending on image count)

**Output:**
```
✅ Total images loaded: 4
✅ DATASET CREATION COMPLETE!
📦 Your .pt file is ready: ./my_dataset.pt
```

---

### Step 2: Load & Test Dataset (< 1 minute)

```bash
python step2_load_and_use_dataset.py
```

**What happens:**
- Loads `my_dataset.pt`
- Shows data shapes
- Demonstrates how to use data

**Output:**
```
📂 Loading dataset from: ./my_dataset.pt
✅ Dataset loaded successfully!
   Total images: 4
   Image shape: torch.Size([4, 3, 416, 416])
```

---

### Step 3: Train Model (5-30 minutes depending on data)

```bash
python step3_complete_training_example.py
```

**What happens:**
- Creates neural network
- Trains for 2 epochs
- Saves trained model as `trained_model.pth`
- Runs inference on all images

**Output:**
```
Epoch 1/2
   Batch 1: Loss = 0.5234
   ...
✅ Model saved to: ./trained_model.pth
✅ COMPLETE PIPELINE FINISHED!
```

## 💻 Using Dataset in Your Own Code

### Method 1: Simple 3-line loading
```python
import torch

# Load dataset
data = torch.load('./my_dataset.pt')
images = data['images']  # Shape: [N, 3, 416, 416]
labels = data['labels']  # List of N labels
```

### Method 2: With DataLoader (Recommended)
```python
from step2_load_and_use_dataset import load_dataset_with_dataloader

dataloader, dataset = load_dataset_with_dataloader(
    './my_dataset.pt',
    batch_size=32,
    shuffle=True
)

# Use in training loop
for batch in dataloader:
    images = batch['image']
    labels = batch['label']
    # Your model training code here
```

### Method 3: Load trained model
```python
import torch
from step3_complete_training_example import SimpleDetectionModel

# Load model
model = SimpleDetectionModel()
model.load_state_dict(torch.load('./trained_model.pth'))
model.eval()

# Use for predictions
with torch.no_grad():
    predictions = model(images)
```

## ⚙️ Common Customizations

### Change image size from 416 to 640:
```python
# In step1_create_dataset_pt.py, line 20:
self.image_size = (640, 640)  # Change this
```

### Change batch size:
```python
# In step2 or step3:
dataloader = load_dataset_with_dataloader(
    './my_dataset.pt',
    batch_size=64  # Change from 32 to 64
)
```

### Change number of training epochs:
```python
# In step3_complete_training_example.py, line 248:
NUM_EPOCHS = 10  # Change from 2 to 10
```

### Use different model:
```python
# In step3_complete_training_example.py, replace SimpleDetectionModel with:
from torchvision.models import resnet50
model = resnet50(pretrained=True)
```

### Use GPU:
```python
# Automatic detection (add to any file):
device = 'cuda' if torch.cuda.is_available() else 'cpu'
```

## 🔍 Understanding Output Shapes

```
When you load your dataset:
├── images:  [4, 3, 416, 416]
│           ├── 4 = number of images
│           ├── 3 = RGB channels
│           ├── 416 = height
│           └── 416 = width
│
├── labels: ['0 0.5 0.5 0.3 0.3', ...] (one per image)
│
└── image_paths: Metadata about each image
```

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| "No images found" | Check `dataset/train/images` has .jpg or .png files |
| "ModuleNotFoundError: torch" | Run: `pip install torch torchvision` |
| "CUDA out of memory" | Reduce batch_size from 32 to 8 or 4 |
| "File not found: dataset" | Make sure you're in the right directory |
| Large file size (300+ MB) | Normal for 1000+ images. Can compress if needed |

## 📊 Data Format Details

Your dataset folder should look like:
```
dataset/
├── train/
│   ├── images/
│   │   ├── Cars0_png.rf.xxx.jpg
│   │   ├── Cars1_png.rf.xxx.jpg
│   │   └── ...
│   └── labels/
│       ├── Cars0_png.rf.xxx.txt
│       ├── Cars1_png.rf.xxx.txt
│       └── ...
└── valid/
    ├── images/
    │   └── ...
    └── labels/
        └── ...
```
