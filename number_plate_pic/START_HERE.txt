Your dataset folder should have:
  dataset/
  ├── train/
  │   ├── images/    (your training images)
  │   └── labels/    (corresponding .txt label files)
  └── valid/
      ├── images/    (validation images)
      └── labels/    (validation labels)

================================================================================
                      ❓ FAQ
================================================================================

Q: Which script should I run first?
A: Run quick_start.py (does everything) OR step1_create_dataset_pt.py

Q: What if I want to use my own model?
A: Replace SimpleDetectionModel in step3 with your model

Q: Can I change image size?
A: Yes, edit step1_create_dataset_pt.py line 20: self.image_size = (640, 640)

Q: How do I use this with YOLO?
A: Create .pt file, then use with: model.train(data='data.yaml')

Q: What's the .pt file format?
A: Dictionary containing:
   - 'images': tensor [N, 3, 416, 416]
   - 'labels': list of labels
   - 'metadata': info about dataset

Q: File size too large?
A: Normal. 1000 images ≈ 200-300 MB. Use compression if needed.

================================================================================
                      📞 TROUBLESHOOTING
================================================================================

Error: "No images found"
  → Check dataset/train/images/ has .jpg/.png files
  → Verify file extensions are lowercase

Error: "ModuleNotFoundError: torch"
  → Run: pip install torch torchvision

Error: "CUDA out of memory"
  → Reduce batch_size (32 → 8 or 4)
  → Or use CPU instead

Error: "File not found"
  → Run from the ANPR- folder
  → Check dataset/ folder exists
